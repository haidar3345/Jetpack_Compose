# Jetpack Compose Tutorial Project

This is a modern Android application built with Jetpack Compose, showcasing a conversation list with expandable message cards.

## Features
- **Jetpack Compose UI**: Built entirely with declarative UI components.
- **Dynamic Content**: Displays a list of messages using `LazyColumn`.
- **Interactive Elements**: Expandable message cards with animations (`animateColorAsState`, `animateContentSize`).
- **Material 3 Design**: Uses Material 3 components and theming.
- **Dark Mode Support**: Fully compatible with Light and Dark modes.

## Getting Started
1. Clone the repository.
2. Open the project in Android Studio (Ladybug or newer).
3. Ensure you have Android SDK 37 (API 37) installed, as required by the latest dependencies.
4. Build and run the project on an emulator or physical device.
